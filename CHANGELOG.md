# Changelog

Všechny významné změny v projektu Nouri budou zaznamenány v tomto souboru.

## [0.1.0] - 2026-03-16

### Přidáno
- **Restrukturalizace aplikace**: Rozdělení do modulárních složek (`components`, `features`, `context`, `db`).
- **Lokální databáze**: Implementace Dexie.js pro offline-first ukládání jídel v prohlížeči.
- **Systém témat**: Podpora pro Světlý, Tmavý a Systémový režim s přepínačem v profilu.
- **iOS Optimalizace**: Použití `h-dvh` pro iPhony s výřezem a plynulé scrollování s pevnou navigací.
- **PWA Podpora**: Konfigurace pro instalaci na plochu iPhonu.
- **Nové sekce**: Domovská obrazovka (Home), Statistiky (Stats), Recepty (Recipes) a Profil (Profile).
- **Design**: Nasazení Lucide React ikon a Tailwind CSS v4 s podporou Dark Mode.
- **Dokumentace**: Kompletně přepsán README.md.
